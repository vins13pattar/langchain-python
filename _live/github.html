1. Understanding Git and GitHub

Tool Purpose
Git Tracks code changes and maintains version history
GitHub Cloud platform that hosts Git repositories
GitHub Desktop GUI application for working with Git
Your Computer ↓ GitHub Desktop ↓ Local Repository ↓ Push / Pull ↓ GitHub Repository

2. Install GitHub Desktop

STEP 1

Download GitHub Desktop:

https://desktop.github.com

Install and launch the application.

3. GitHub Authentication

STEP 2
  1. Open GitHub Desktop
  2. Click Sign in to GitHub.com
  3. Browser opens
  4. Login to GitHub
  5. Authorize GitHub Desktop
  6. Return to application
GitHub Desktop uses OAuth authentication.

How OAuth Works

GitHub Desktop ↓ Browser Login ↓ GitHub OAuth ↓ Access Token ↓ Stored Securely

Authentication Methods

Method Recommended
OAuth (GitHub Desktop) ✅ Yes
Personal Access Token ✅ Learn Later
SSH Keys ✅ Professional Teams
Password ❌ Unsupported

STEP 3

5. Publish Repository to GitHub

STEP 4

  • Click Publish

14. Daily Developer Workflow

1. Fetch Origin ↓ 2. Pull Origin ↓ 3. Create Branch ↓ 4. Write Code ↓ 5. Commit ↓ 6. Push Origin ↓ 7. Create Pull Request

15. Real Industry Workflow

Clone Repository ↓ Create Feature Branch ↓ Develop Feature ↓ Commit Changes ↓ Push Branch ↓ Create Pull Request ↓ Code Review ↓ Merge to Main ↓ Pull Latest Main ↓ Deployment
